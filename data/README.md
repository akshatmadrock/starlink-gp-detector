# Data

The dataset is **not tracked in git** due to size (~86 MB).

## Source

Generated by [Starlink Sentinel](https://github.com/ece257b/starlink-passive-detection) —
a passive Ku-band SDR receiver running continuously on the UCSD rooftop from
February 20 to March 21, 2026.

## Getting the data

Place `detections.db` at `data/detections.db` (relative to the repo root),
or update `data.db_path` in `configs/config.yaml` to point to your copy.

The database is available from the companion repo:
`../starlink-passive-detection/data/detections.db`

## Schema

| Column | Type | Description |
|---|---|---|
| `id` | int | Row ID |
| `timestamp_utc` | str | UTC timestamp of the 30-second window |
| `unix_time` | float | Unix timestamp |
| `z_score` | float | Welch PSD z-score at 750 Hz (primary detection statistic) |
| `peak_freq_hz` | float | Actual peak frequency detected (expected ~750 Hz) |
| `harmonics` | int | Number of harmonics confirmed (0–3) |
| `sat_name` | str | Highest-elevation Starlink satellite via TLE matching |
| `sat_elevation` | float | Satellite elevation angle in degrees |
| `confidence` | str | `HARD` (z≥4.0), `SOFT` (z≥3.0 + harmonic), or `BACKGROUND` |
